# Git & GitHub 新手教程

从零开始学习 Git版本控制和 GitHub协作的完整教程。

## 在线访问

https://mrifox.github.io/git-tutorial/

## 教程内容

| 章节 | 内容 | 预计时间 |
|------|------|----------|
| 1. 安装Git | 下载安装Git，配置环境变量 | 10分钟 |
| 2. Git基础概念 | 仓库、提交、分支、远程的概念 | 15分钟 |
| 3. 基本操作 | init、add、commit、status命令 | 20分钟 |
| 4. 分支管理 | branch、checkout、merge命令 | 20分钟 |
| 5. 远程操作 | remote、push、pull、clone命令 | 20分钟 |
| 6. 解决冲突 | merge conflict的处理方法 | 15分钟 |
| 7. 撤销操作 | reset、revert、stash命令 | 15分钟 |
| 8. GitHub入门 | 注册账号、创建仓库 | 15分钟 |
| 9. README与LICENSE | 项目说明文件和开源协议 | 10分钟 |
| 10. Issue管理 | 创建Issue、标签、里程碑 | 15分钟 |
| 11. Pull Request | 提交PR、代码审查流程 | 20分钟 |
| 12. GitHub Pages | 部署静态网站到GitHub | 15分钟 |
| 13. Fork与协作 | 开源项目贡献流程 | 15分钟 |
| 附录：必备英语 | Git/GitHub常用术语 | 15分钟 |

## 功能特点

- 13个章节 + 附录，覆盖Git和GitHub核心功能
- 学习进度自动保存（localStorage）
- 5套题库随机抽取的毕业考试
- 毕业证书（90分以上获得）
- 响应式设计，支持手机访问
- GitHub界面教学插图
- 趣味对话框和代码示例

## 技术栈

- 纯HTML/CSS/JavaScript
- localStorage保存学习进度
- GitHub Pages部署

## 本地运行

直接双击 `index.html` 文件即可在浏览器中打开。
