5. 远程操作

☁️
本地的代码需要推送到云端备份和分享，这就是远程操作！

添加远程仓库

# 添加远程仓库（通常命名为origin）
git remote add origin https://github.com/用户名/仓库名.git

# 查看远程仓库
git remote -v

推送到远程

# 首次推送（设置上游分支）
git push -u origin main

# 之后推送
git push

从远程拉取

# 拉取远程更新
git pull

# 只下载不合并
git fetch

克隆仓库

下载别人的项目到本地：

# 克隆远程仓库
git clone https://github.com/用户名/仓库名.git

# 克隆到指定文件夹
git clone https://github.com/用户名/仓库名.git my-folder

常用命令速查

命令 作用
git remote add 添加远程仓库
git push 推送到远程
git pull 从远程拉取
git clone 克隆仓库
