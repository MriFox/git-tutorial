13. Fork与协作

🍴
Fork是参与开源项目的起点，让你可以在别人的项目上自由修改！

什么是Fork？

Fork是把别人的仓库复制一份到你自己的GitHub账号下，这样你可以：

  • 自由修改代码
  • 不影响原项目
  • 提交PR贡献给原项目

参与开源项目的流程

  1. Fork原项目到你的账号
  2. Clone你的Fork到本地
  3. 创建新分支开发功能
  4. 提交代码并推送到你的Fork
  5. 在GitHub上创建PR到原项目
  6. 等待原作者审查和合并
# 克隆你的Fork
git clone https://github.com/你的用户名/原项目名.git

# 添加原项目为upstream
git remote add upstream https://github.com/原作者/原项目名.git

# 同步原项目的更新
git fetch upstream
git merge upstream/main

开源贡献礼仪

1. 先阅读项目的CONTRIBUTING.md
2. 从简单的Issue开始（如good first issue）
3. 遵循项目的代码风格
4. 保持PR小而专注
