3. 基本操作

⚡
掌握这几个命令，你就能开始用Git管理代码了！

初始化仓库

把一个文件夹变成Git仓库：

# 进入项目文件夹
cd my-project

# 初始化Git仓库
git init

查看状态

查看哪些文件被修改了：

git status

输出说明：

  • 绿色 - 已暂存，准备提交
  • 红色 - 已修改，未暂存
  • Untracked - 新文件，未被Git跟踪

添加到暂存区

# 添加指定文件
git add filename.txt

# 添加所有修改
git add .

# 添加所有txt文件
git add *.txt

提交到仓库

# 提交并添加描述
git commit -m "描述你做了什么修改"

# 示例
git commit -m "添加了登录功能"

提交信息规范

好的提交信息应该简洁明了，说明"做了什么"而不是"怎么做的"

查看历史

# 查看提交历史
git log

# 简洁显示
git log --oneline

常用命令速查

命令 作用
git init 初始化仓库
git status 查看状态
git add 添加到暂存区
git commit 提交到仓库
git log 查看历史
