1. 安装Git

🎉
恭喜你迈出第一步！安装Git就像给你的电脑装上"时间机器"，让你的代码可以随时回到过去！

什么是Git？

Git是一个版本控制系统，它可以：

  • 记录代码的每一次修改
  • 让你随时回到之前的版本
  • 多人同时修改同一个项目
  • 管理项目的不同版本（分支）

简单理解：Git = 代码的"后悔药"

改错了？没关系，Git帮你回到之前的状态！

Windows安装

  1. 访问 https://git-scm.com/download/win 下载安装包
  2. 运行安装程序，一路点击"Next"即可
  3. 安装完成后，右键桌面空白处，看到"Git Bash Here"就成功了
💡 安装时建议选择默认选项，特别是要勾选"Add to PATH"（通常默认已勾选）。

Mac安装

# 方法1：使用Homebrew（推荐）
brew install git

# 方法2：安装Xcode Command Line Tools
xcode-select --install

Linux安装

# Ubuntu/Debian
sudo apt install git

# CentOS/RHEL
sudo yum install git

验证安装

打开终端（Windows打开Git Bash），输入：

git --version

如果显示版本号（如 git version 2.45.0），说明安装成功！

配置用户信息

安装完成后，先配置你的身份信息：

# 配置用户名
git config --global user.name "你的名字"

# 配置邮箱
git config --global user.email "你的邮箱@example.com"
⚠️ 这里的邮箱最好和你注册GitHub时使用的邮箱一致！
