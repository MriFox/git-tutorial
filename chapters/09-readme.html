9. README与LICENSE

📖
好的项目需要好的"门面"，README就是项目的"说明书"！

README.md

README是项目的说明文件，通常包含：

  • 项目名称和简介 - 这个项目是做什么的
  • 安装方法 - 如何使用这个项目
  • 使用示例 - 展示项目功能
  • 贡献指南 - 如何参与项目开发
# 项目名称

## 简介
这是一个xxx项目

## 安装
```bash
npm install
```

## 使用
```bash
npm start
```

LICENSE文件

LICENSE是开源协议，规定别人可以怎么使用你的代码。

协议 特点
MIT 最宽松，可商用
Apache 2.0 宽松，需保留版权
GPL 严格，衍生项目也必须开源
💡 不知道选哪个？MIT协议是目前最流行的选择！
