使用Git和GitHub必备英语

📚
Git命令和GitHub界面都是英文的，掌握这些词汇能让你更顺畅地使用！

Git命令词汇

英语 中文意思
commit 提交
branch 分支
merge 合并
push 推送
pull 拉取
clone 克隆
stash 暂存
revert 回退

GitHub界面术语

英语 中文意思
repository 仓库
fork 复刻
pull request 拉取请求
issue 议题
release 发布
star 收藏
watch 关注

常见错误信息

错误信息 意思
permission denied 权限被拒绝
repository not found 找不到仓库
connection refused 连接被拒绝
merge conflict 合并冲突
